## Examples

### Size

There are 11 sizes to choose from. Each size handles its own `font-size` and `letter-spacing` accordingly.

```js live
<Text as="p" mt={1} size={0}>Text component</Text>
<Text as="p" mt={1} size={1}>Text component</Text>
<Text as="p" mt={1} size={2}>Text component</Text>
<Text as="p" mt={1} size={3}>Text component</Text>
<Text as="p" mt={1} size={4}>Text component</Text>
<Text as="p" mt={1} size={5}>Text component</Text>
<Text as="p" mt={1} size={6}>Text component</Text>
<Text as="p" mt={1} size={7}>Text component</Text>
<Text as="p" mt={1} size={8}>Text component</Text>
<Text as="p" mt={1} size={9}>Text component</Text>
<Text as="p" mt={1} size={10}>Text component</Text>
```

### Font weight

There are 3 weight options to choose from.

```js live
<Text as="p" fontWeight="500">Text component</Text>
<Text as="p" fontWeight="600">Text component</Text>
<Text as="p" fontWeight="700">Text component</Text>
```

### Text truncation

When `true` this will truncate your text based on its maximum width.

```js live
<Box width={100}>
  <Text as="p" truncate>
    At Containous, our goal is to make networking boring.
  </Text>
</Box>
```
